## 💻 Troubleshooting Tips

If you face issues, try these solutions:

- **App does not start:** Restart your computer and try again.
- **Failed to add a session:** Check your internet connection.
- **Agent responses are delayed:** Make sure your network is stable.
- **Installation failed:** Run the installer as Administrator.

Check the logs folder inside the ccbox app directory for error details.
